Internet X-mas Tree Project (Dec 2015)

For Christmas last year, we decorated our tree with a 20m strip of individually-addressable RGB leds. I first made a python program to change the colors and patterns of the tree, then a web interface to control it. I didn't know any PHP before this, so I learned enough in the time of a few hours. I also wasn't familiar with Git before this, so I took it upon myself to learn that as well.
Project avaliable on my GitHub page.

GBJAM 4 - Low Battery (August 2015)

Made for GBJAM 4 a "game jam" themed around the style of the original GameBoy handheld. Like any other "game jam", everything had to be made from scratch, but this event had to be GameBoy themed. It must have the original GameBoy resolution of 160px x 144px, and use only 4 colors. For this project, I learned the MonoGame game engine, which is a open source implementation of the Microsoft XNA 4.x framework. While we were given 10 days for this game, I was on vacation for most of it. The game turned out to be very short, due to a lack of levels.
Check it out here.

Ludum Dare 32 - Operation Peeps (April 2015)

Ludum Dare is a "game jam" event where solo participants have 48 hours to make an entire game based off of the given theme. All of the code and art assets must be created from scratch. I took this event as an opportunity to better familiarise myself with the Unity3D game engine (so that I could later use it for Hunt the Wumpus).
My entry can be found here.

Hunt the Wumpus (Feb - May 2015)

Hunt the Wumpus was an event where groups were assigned to work on their own "Hunt the Wumpus" games, based only off of the given criteria. My team placed 1st runner up in the competition. Our game used the Unity3D game engine, and was coded in C#. It also supported controller and touchscreen support.
I made a website to show it off. While it has screenshots, you'll need the Unity Web Player to play it in-browser.

Snake Game (Spring 2015)

This project I made for my 2nd semester of Computer Science and Engineering. It's a 8x8 LED display and a controller with 6 buttons. I then programmed a game of snake to run on the Arduino, complete with a menu and high-score tracking (until it's turned off). The main controller board was custom made, designed in Fritzing and etched.

Raspberry Pi Clock

Around the time when that kid got in trouble for bringing his disassembled clock to school, I decided it would be interesting to make my own spin on it. Using a Raspberry Pi, 16x2 LCD display and a real-time clock module (not shown), I created a clock which kept the time, even when the device was turned off. It also made it clear to anyone around that it was not a bomb.
