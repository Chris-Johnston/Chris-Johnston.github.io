Ludum Dare 32 (2015)

Ludum Dare is a "game jam" event where solo participants have 48 hours to make an entire game based off of the given theme. All of the code and art assets must be created from scratch. I took this event as an opportunity to better familiarise myself with the Unity3D game engine (so that I could later use it for Hunt the Wumpus). My entry can be found here.

Hunt the Wumpus (2015)

Hunt the Wumpus was an event where groups were assigned to work on their own "Hunt the Wumpus" games, based only off of the given criteria. My team placed 1st runner up in the competition. Our game used the Unity3D game engine, and was coded in C#. It also supported controller and touchscreen support. I made a website to show it off.
While it has screenshots, you'll need the Unity Web Player to play it in-browser.
